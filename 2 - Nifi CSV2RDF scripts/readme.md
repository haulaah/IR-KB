# Apache NiFi ExecuteScript Processor: CSV to RDF Conversion

A custom Apache NiFi "ExecuteScript" processor script that converts the IR Log CSV data into RDF format according to CIRPO ontology (https://haulaah.github.io/CIRPO/)

The groovy script reads CSV input, transforms rows into RDF triples, and outputs structured semantic data.

## Features

- Converts CSV records into RDF triples
- Designed for Apache NiFi "ExecuteScript" processor
- Output to a format ready to be loaded to GraphDB


Latest version: v10
